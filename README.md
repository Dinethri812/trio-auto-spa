A full-stack MERN application for a vehicle detailing business.  
Customers can submit service bookings, and the admin can view booking statistics via a dashboard.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Customer booking form
- Booking data stored in MongoDB
- Dashboard showing total and pending bookings

## How to Run
1. Run MongoDB
2. Start backend: `npx nodemon server.js`
3. Start frontend: `npm start`
